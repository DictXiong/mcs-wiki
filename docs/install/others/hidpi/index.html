      高分屏设置 #

      在电脑的显示设置中可找到“缩放”。如果它高于100%，你的屏幕就可能是高分屏。

      如果你的电脑是高分屏，且“缩放”在150%以上，建议进行以下配置，可以提高性能。

      在Java安装目录下 bin 文件夹中找到java.exe（或者javac.exe，取决于HMCL中配置使用哪个；如果不确定，二者都配置）。右键 – 属性 – 兼容性 – 更改高DPI设置 – 替代高DPI缩放行为，更改为“系统（增强）”。如下图所示。
